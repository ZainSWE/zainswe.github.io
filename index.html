Coin Toss Simulator

Coin Toss Simulator is my first JavaScript web-app that allows users to settle scores by flipping a virtual coin with heads/tails. Some features that this application includes are the following:

  • Auto Toss
  • Reset Score
  • Counting Scores
  • 3D Coin Animation
  • 5x Streak Animation

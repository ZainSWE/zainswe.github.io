LATEST PROJECT:

Coin Toss Simulator

Coin Toss Simulator is my first JavaScript web-app that allows users to settle scores by flipping a virtual coin with heads/tails. Some features that this application includes are the following:

  • Auto Toss
  • Reset Score
  • Counting Scores
  • 3D Coin Animation
  • 5x Streak Animation

Project: GPYOU

1st place for Best UI @ Google Developer Student Clubs (GDSC) 2025 Guelph Hackathon.

PROBLEM:

The graphics card market is constantly changing - dozens of models, confusing specifications, and widely varying prices make it difficult to know what to choose.

SOLUTION:

GPYOU simplifies that process by asking you 4 questions, then offering personalized recommendations tailored to you.

FRAMEWORK:

HTML / CSS / JS / Google Gen Ai API
